# Character Management Workspace

Full-stack character management application with:

- `frontend/`: Next.js 16, React 19, GraphQL Code Generator, React Query, `nuqs`, and Axios
- `backend/`: NestJS 11, GraphQL, Prisma ORM, and PostgreSQL

## Environment (Already Wired)

The backend is configured for:

- Database name: `character_management_db`
- Database user: `postgres`
- Database password: `YOUR_PASSWORD`
- Backend URL: `http://localhost:3001/graphql`
- Frontend URL: `http://localhost:3000`

The local env files are already included for convenience:

- `backend/.env`
- `frontend/.env.local`

Examples are also included:

- `backend/.env.example`
- `frontend/.env.local.example`

If you want to change credentials, update:

- `backend/.env` → `DATABASE_URL=postgresql://postgres:<PASSWORD>@localhost:5432/character_management_db?schema=public`
- `docker-compose.yml` → `POSTGRES_PASSWORD: <PASSWORD>`

Make sure both values match.

## Step 1: Run PostgreSQL

Option 1: Use your local PostgreSQL server and create a database named `character_management_db`.

Option 2: Use Docker Compose (recommended if you do not already have PostgreSQL running):

```bash
docker compose up -d
```

## Step 2: Install Dependencies

Install packages in each project and at the root:

```bash
npm install
npm --prefix backend install
npm --prefix frontend install
```

If PowerShell blocks `npm` scripts on your machine, use `npm.cmd` instead:

```powershell
npm.cmd install
npm.cmd --prefix backend install
npm.cmd --prefix frontend install
```

## Step 3: Create Tables With Prisma (Migration + Seed)

Generate the Prisma client, create the `Character` table, and seed sample data:

```bash
npm run db:generate
npm run db:migrate
npm run db:seed
```

PowerShell equivalent:

```powershell
npm.cmd run db:generate
npm.cmd run db:migrate
npm.cmd run db:seed
```

## Step 4: Generate Frontend GraphQL Hooks

```bash
npm run codegen
```

## Step 5: Start the Apps

Run both apps together:

```bash
npm run dev
```

Or start them separately:

```bash
npm --prefix backend run start:dev
npm --prefix frontend run dev
```

PowerShell equivalents:

```powershell
npm.cmd run dev
npm.cmd --prefix backend run start:dev
npm.cmd --prefix frontend run dev
```

## Useful Commands

```bash
npm run build
npm run lint
npm run test
```


## Feature Summary

- GraphQL query for fetching characters
- Server-side filtering by `status`, `gender`, and search text
- Prisma seeder with varied character data
- Typed frontend GraphQL hooks generated by GraphQL Code Generator
- React Query data fetching and caching
- URL-synced filter state using `nuqs`
- Responsive UI with loading, empty, and error states
